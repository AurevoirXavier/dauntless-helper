## Features

#### cpp-ver-imgui-ver

- **F1**: focus on menu - 激活菜单
- **F10**: auto farm - 自动打怪
- **Home**: show/hide menu - 显示/隐藏菜单

- **Other feature**: use **Tab** to modify the slide bar value - 使用 Tab 键可修改滑动条数值
